# DP-DL Security Survey Summary

This repository provides a summary of the paper ***“Attacks and Defenses in Differentially Private Deep Learning: New Security Risks in New Era”***. 

The document summarizes the key ideas of the paper, including:

* The background of **Differentially Private Deep Learning (DP-DL)**
* Security risks that may arise in DP-DL systems
* A classification of risks at the **data, algorithm, and architecture levels**
* Typical attacks and corresponding defense strategies
* Key conclusions and future research directions

The document is intended to serve as a **concise reading note and structured overview** of the paper for quick understanding and reference.
